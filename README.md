# 3D Portfolio Website

A 3D immersive portfolio website built with React, Three.js, and Framer Motion.

## Features

- **3D Floating Islands**: Navigate through different sections of the portfolio in a 3D space.
- **Interactive Elements**: Hover effects and smooth animations.
- **Responsive Design**: Works on desktop and mobile.
- **Content from Resume**: Includes Experience, Projects, Skills, and Contact sections.

## Technologies Used

- **React** (Vite)
- **Three.js** (@react-three/fiber)
- **@react-three/drei** (Helpers)
- **Framer Motion** (Animations)
- **Tailwind CSS** (Styling - utility classes used in components)

## Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Customization

- Update `src/constants/index.js` with your own data.
- Modify `src/components/PortfolioScene.jsx` to change 3D objects.
